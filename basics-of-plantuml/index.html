Einführung in PlantUML

PlantUML ist eine Open-Source-Sprache zur Erstellung von UML-Diagrammen. Sie verwendet eine einfache Textsyntax, um verschiedene Arten von Diagrammen zu erstellen.

Grundlegende Syntax

Die grundlegende Syntax für ein Flowchart in PlantUML sieht wie folgt aus:

@startuml
start
:Erste Aktion;
:Zweite Aktion;
if (Entscheidung?) then (Ja)
  :Dritte Aktion;
else (Nein)
  :Vierte Aktion;
endif
stop
@enduml
