HTML Grundlagen

HTML ist die Sprache zum Strukturieren von Webseiten.

HTML Beispiel:

<!DOCTYPE html>
<html lang="de">
<head>
    <meta charset="UTF-8">
    <title>Beispiel</title>
</head>
<body>
    <h1>Einführung in HTML</h1>
</body>
</html>

CSS Grundlagen

CSS gestaltet das Aussehen von HTML-Dokumenten.

CSS Beispiel:

body {
    font-family: Arial, sans-serif;
    background-color: #f0f0f0;
    color: #333;
}
h1 {
    color: #0056b3;
}
