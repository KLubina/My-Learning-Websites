HTML Grundlagen

HTML (Hypertext Markup Language) ist die standardisierte Markup-Sprache für Dokumente, die im Web angezeigt werden sollen.

HTML Beispiel:

<!DOCTYPE html>
    <html lang="de">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>HTML Grundlagen</title>
    </head>
    <body>
        <h1>Willkommen zu HTML</h1>
    </body>
    </html>

CSS Grundlagen

CSS (Cascading Style Sheets) wird verwendet, um das Aussehen und Layout von HTML-Dokumenten zu gestalten.

CSS Beispiel:

body {
        font-family: Arial, sans-serif;
        background-color: #f0f0f0;
        color: #333;
    }
    h1 {
        color: #0056b3;
    }
